# 疑难解答

如果你使用 VuePress-Creator 出现问题时，可以来 Github 提 issue 哦！

Q： 正常配置了 @vuepress/back-to-top 插件仍然无法使用？终端提示：warning An error was encountered in plugin “@vuepress/back-to-top” ？
A： 请尝试在项目下安装 vuepress 依赖：

yarn add -D vuepress
# 或者 npm install -D vuepress

Q： 为什么开启了 lastUpdated 功能，文档底部还是没有显示信息？
A： lastUpdated 是基于每个文件最后一次 git 提交的 UNIX 时间戳(ms)，意味着你的项目应该是基于 git 的项目，它才能正常显示时间。
